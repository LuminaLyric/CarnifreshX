CarnifreshX – Fresh Meat Shop Website

A modern and responsive meat shop landing page built using HTML, Tailwind CSS, and JavaScript. CarnifreshX is designed to showcase fresh meat products, pricing, and business information in a clean and responsive interface.

Features
Responsive design for mobile, tablet, and desktop
Fixed transparent navigation bar
Mobile hamburger menu
Smooth scrolling navigation
Hero section with background image
About section
Meat pricing table
Contact section
Clean UI using Tailwind CSS
Tech Stack
HTML5
Tailwind CSS
JavaScript
